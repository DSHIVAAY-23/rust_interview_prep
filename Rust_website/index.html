🔑 The Core Insight
Every Rust concept is just one of three rules applied to a different context: &T (read), &mut T (mutate), or T (own). When you understand this, every compiler error makes sense immediately.

The one mental check
Before writing any Rust code, ask: "Does this need to read, mutate, or own the data?"

&T — read, cheapest, many at once
&mut T — exclusive write, one at a time
T — moves data, Drop at end
Interview anchor
"Why Arc?" → multiple owners need to read T across threads
"Why Mutex?" → need to mutate T across threads
"Why FnOnce?" → closure owns T — can only run once
"Why iter_mut()?" → need &mut T

DSA Pattern Guide approach
How to identify patterns + approach templates for each
Pattern Identification — Decision Tree
Is window/subarray size fixed? → Sliding Window fixed
Need to find two elements in sorted array? → Two Pointers
Need max/min sum subarray? → Kadane
Need count subarrays with exact sum? → Prefix Sum + HashMap
Linked list cycle or middle? → Fast & Slow Pointers
Sorted array, find value? → Binary Search
Answer space — minimize/maximize? → Binary Search on Answer
Tree — path/depth/single node? → DFS recursive
Tree — level by level? → BFS + queue
Two Pointers — "squeeze from both ends"
Sorted array, find two elements satisfying a condition. Left + right pointers, squeeze inward based on comparison.
let (mut l, mut r) = (0, nums.len() - 1);
while l < r {
    let sum = nums[l] + nums[r];
    match sum.cmp(&target) {
        Equal   => { /* found */ }
        Less    => l += 1,   // sum too small → move left right
        Greater => r -= 1,   // sum too big → move right left
    }
}
// Time: O(n)  Space: O(1)
Sliding Window — "expand and shrink"
Fixed: add right, remove left (i - k). Variable: expand right, shrink left when condition violated.
// Variable window — smallest subarray with sum ≥ target
let (mut l, mut sum) = (0, 0);
for r in 0..nums.len() {
    sum += nums[r];                      // expand right
    while sum >= target {               // shrink left
        ans = ans.min(r - l + 1);
        sum -= nums[l]; l += 1;
    }
}
// Key: right pointer never goes back → O(n)
Kadane (max subarray) + Prefix Sum (exact sum count)
// Kadane — max subarray sum
let (mut max_sum, mut curr) = (nums[0], nums[0]);
for &n in &nums[1..] {
    curr = n.max(curr + n);  // extend or fresh start?
    max_sum = max_sum.max(curr);
}

// Prefix Sum — count subarrays with sum = k
let mut map = HashMap::new();
map.insert(0, 1);  // empty prefix — DON'T FORGET THIS
let (mut prefix, mut count) = (0, 0);
for &n in &nums {
    prefix += n;
    count += map.get(&(prefix - k)).unwrap_or(&0);
    *map.entry(prefix).or_insert(0) += 1;
}
Binary Search — classic + answer space
// Classic — find in sorted array
let (mut lo, mut hi) = (0, nums.len() - 1);
while lo <= hi {
    let mid = lo + (hi - lo) / 2;  // usize overflow safe
    match nums[mid].cmp(&target) {
        Equal   => return mid as i32,
        Less    => lo = mid + 1,
        Greater => hi = mid - 1,
    }
}

// Answer space — minimize X such that feasible(X) is true
let (mut lo, mut hi) = (1, *piles.iter().max().unwrap());
while lo < hi {                     // lo < hi, not <=
    let mid = lo + (hi - lo) / 2;
    if feasible(mid) { hi = mid; }  // mid works, try smaller
    else             { lo = mid + 1; }
}  // lo is the answer
Trees — DFS and BFS templates
// DFS — path, depth, single node answer
fn dfs(node: Option<&TreeNode>) -> i32 {
    let node = match node { Some(n) => n, None => return 0 };
    let left  = dfs(node.left.as_deref());
    let right = dfs(node.right.as_deref());
    left.max(right) + 1   // current node logic here
}

// BFS — level order
let mut queue = VecDeque::from([root]);
while let Some(node) = queue.pop_front() {
    // process node
    if let Some(l) = node.left  { queue.push_back(l); }
    if let Some(r) = node.right { queue.push_back(r); }
}
Graphs — DFS and BFS templates
// DFS — visited set prevents cycles
fn dfs(node: usize, adj: &Vec<Vec<usize>>,
       visited: &mut HashSet<usize>) {
    if !visited.insert(node) { return; }  // already visited
    for &nb in &adj[node] { dfs(nb, adj, visited); }
}

// BFS — shortest path
let mut queue = VecDeque::from([start]);
let mut visited = HashSet::from([start]);
let mut dist = 0;
while !queue.is_empty() {
    for _ in 0..queue.len() {
        let node = queue.pop_front().unwrap();
        for &nb in &adj[node] {
            if visited.insert(nb) { queue.push_back(nb); }
        }
    }
    dist += 1;
}
